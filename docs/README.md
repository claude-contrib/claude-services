## MCP Servers

Claude Code uses the [Model Context Protocol](https://modelcontextprotocol.io) to connect external tools. Each service plugin provides a `.mcp.json` that configures the server command and arguments.

- [MCP documentation](https://modelcontextprotocol.io/docs) — Protocol specification and server development
